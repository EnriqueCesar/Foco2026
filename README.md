# FOCO 2026 V6.1

Dashboard ejecutivo estilo Starbucks para gestión FOCO nacional.

## Vistas
- RD: Mes | Región, ranking de DMs y tendencia semanal.
- DM: Mes | DM, ranking de tiendas y tendencia semanal.
- Tienda: Mes | Buscar tienda o CeCo, FOCO con acciones por pilar.

## Publicación
Subir `index.html`, `style.css`, `app.js`, `data.js` y `README.md` a GitHub Pages desde la rama `main` en `/root`.
