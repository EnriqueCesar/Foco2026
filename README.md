# FOCO 2026 · GitHub V6

Versión ejecutiva para GitHub Pages.

## Ajustes V6
- RD: filtro independiente Mes | Región.
- DM: filtro independiente Mes | DM con todos los DMs nacionales.
- Tienda: filtro Mes | buscador inteligente + lista desplegable nacional.
- Vista RD/DM: indicador más compacto y promedio semanal visible por indicador.
- Ranking por promedio mensual.
- Semanas siempre tomadas de Base_Mes_Semana / calendario 2026.
- Tienda conserva formato FOCO con acciones por pilar.

## Uso
1. Sube todos los archivos del ZIP a un repositorio GitHub.
2. Activa GitHub Pages en la rama principal.
3. Abre `index.html`.

## Actualización semanal
Actualizar `data.js` cuando cambie la base nacional.
